O React oferece uma API de contexto que permite criar um contexto compartilhado entre os componentes, eliminando a necessidade de passar props desnecessariamente. Usando a API de contexto, você pode definir um valor no provedor de contexto e acessá-lo nos componentes.
Imagine uma aplicação com vários componentes em conjunto onde você precisa passar uma determinada propriedade para um componente que está no fundo da árvore de componentes. Em vez de passar essa propriedade por todos os componentes intermediários, podemos usar a API de contexto para criar um contexto compartilhado. O valor desejado pode assim ser acessado diretamente pelo componente final sem a necessidade de se propagar por toda a árvore. Isso torna o código mais limpo e legível.
